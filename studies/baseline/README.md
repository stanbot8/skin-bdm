# Baseline Study

Homeostatic epidermis with no wound or fibroblast activation. Validates steady-state tissue turnover, calcium gradient maintenance, and stratification dynamics in unperturbed skin.

## Configuration

- Duration: 12.5 days
- Wound: disabled
- Fibroblasts: disabled

## Usage

```bash
python3 batch/batch.py -n 10 --study baseline --skin normal
```
